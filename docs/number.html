minimalistnumber

mn-number

install

npm install --save minimalist

and bundle it with your preferred tool.

usage

import components in your .js file

import {number} from 'minimalist'

finally in your html, you can use the tag <mn-number>, e.g.

<mn-number></mn-number>

Attributes

The complete list of attributes supported is the following:

placeholder

The title of input, indicate what content the user needs write.

<mn-number placeholder="Type a number"></mn-number>

name

The name is the identifier of input, used to interact with input in javascript

// html <form id="test"> <mn-number placeholder="age" value="27" name="age"></mn-number> </form> // js console.log(test.age) // return the reference to input console.log(test.age.value) // return the value on input console.log(test.data) // will return { age: 27, }

value

Used to set the initial value of mn-number, e.g.

<mn-number value="10"></mn-number>
// you can use value as property in javascript, e.g. const input = document.querySelector('mn-number') input.value = 27 console.log(input.value) // the value always be a number, or undefined

autofocus

Used to automatically focus on input once time page opens.

<mn-number autofocus></mn-number>

required

Set input as required, so user cannot submit form without fill it correctly

<mn-number required></mn-number>

readonly

Sets input as read only, so user cannot change the value, but can navigate to them using keyboard and copy.

<mn-number readonly></mn-number>

disabled

Sets input as disabled, so user cannot navigate to them, but can copy.

<mn-number disabled></mn-number>

precision

Set a mask to number, showing that with precision defined

<mn-number precision='3' value='10'></mn-number>

currency

Set a currency mask to number, with precision 2 by default.

<mn-number currency value='10'></mn-number>
//To change the currency symbol, just use a css variable, e.g. .mn-number { --currency-symbol: 'R$' }

percentage

Set a percentage mask to number, without precision by default.

<mn-number percentage value='1'></mn-number>

max

Define a max value to input. If required, include validation to max value.

<mn-number max='5' required></mn-number>

min

Define a min value to input. If required, include validation to max value.

<mn-number min='5' required></mn-number>

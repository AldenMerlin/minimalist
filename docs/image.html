minimalistimage

mn-image

install

npm install --save minimalist

and bundle it with your preferred tool.

usage

in your html, you can use the tag <mn-image> with src attribute, e.g.

<mn-image src="path/to/image"></mn-image>

If you want a 3d perspective on mouse over, or focus (but only for desktop) just put the component inside a tag

<a> <mn-image src="path/to/image"></mn-image> </a>

Attributes

The complete list of attributes supported is the following:

src

The path relative/absolute to image

<mn-image src="path/to/image"></mn-image>

draggable

Allow to you enable/disable drag image. By default, draggable is enabled.

<mn-image src="path/to/image" draggable='false'></mn-image>

alt

Alternative text to be read by assistive tools, or when image not found.

<mn-image src="path/to/image" alt='some text here'></mn-image>

title

Text to be displayed as title on hover (desktop), or under image (mobile)

<mn-image src="path/to/image" title='some title here'></mn-image>

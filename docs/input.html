minimalistinput

mn-input

install

npm install --save minimalist

and bundle it with your preferred tool.

usage

in your html, you can use the tag <mn-input>, e.g.

<mn-input></mn-input>

Attributes

The complete list of attributes supported is the following:

placeholder

The title of input, indicate what content the user needs write.

<mn-input placeholder="firstname"></mn-input>

name

The name is the identifier of input, used to interact with input in javascript

// html <form id="test"> <mn-input placeholder="firstname" value="John" name="firstname"></mn-input> </form> // js console.log(test.firstname) // return the reference to input console.log(test.firstname.value) // return the value on input console.log(test.data) // will return { firstname: 'John', }

value

Used to set the initial value of mn-input, e.g.

<mn-input value="John"></mn-input>
// you can use value as property in javascript, e.g. const input = document.querySelector('mn-input') input.value = 'John' console.log(input.value)

autofocus

Used to automatically focus on input once time page opens.

<mn-input autofocus></mn-input>

autocapitalize

Only for mobile, by default is off, but you can turn on like below

<mn-input autocapitalize='on'></mn-input>

maxlength

Useful to limit characters on input, e.g.

<mn-input maxlength='3'></mn-input>

pattern

Allow to you validate content using regular expression, in example below, we validate firstname has only letters

<mn-input pattern='^[a-zA-Z]+$'></mn-input>
